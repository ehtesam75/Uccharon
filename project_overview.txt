# Uccharon – Project Overview

## What the Project Does

**Uccharon** is an **AI-powered English learning platform** designed to help users improve their overall English communication skills through interactive conversations, vocabulary building, pronunciation practice, grammar correction, and performance tracking.

The platform works like a **smart AI tutor**, allowing learners to practice English naturally while receiving real-time feedback, personalized learning support, and performance analysis.

The application also includes **Progressive Web App (PWA)** support, allowing users to install and use it like a mobile app across devices.

---

# Core Features

## 1. AI Conversation Practice

* Real-time chat-based English conversation with AI
* Natural conversational learning experience
* Multiple AI provider integration support

## 2. Grammar Correction & Feedback

* Automatic grammar checking for user messages
* Suggests corrected sentence versions
* Provides native speaker style sentence alternatives

## 3. Pronunciation Improvement

* Speech recognition and pronunciation evaluation
* AI feedback on spoken English accuracy
* Speaking practice support for fluency improvement

## 4. Vocabulary Learning System

* Tracks new words learned during conversations
* Daily word learning goals:

  * 50 words/day
  * 100 words/day
  * 150 words/day
  * 200+ words/day
* Vocabulary progress monitoring

## 5. AI Rating & Learning Feedback

* AI rates user responses based on accuracy, fluency, grammar, and clarity
* Provides detailed feedback to help users improve their English
* Daily, weekly, monthly, and all-time performance analysis for learning progress
* Personalized improvement suggestions based on user activity and response quality

## 6. Performance Analytics Dashboard

* Tracks user learning performance over time
* Daily progress statistics
* Word count tracking
* Conversation count tracking
* Learning progress visualization
* Streak tracking system
* Daily and weekly analysis reports for improvement

## 7. Chat Management System

* Automatic new chat creation after login
* Blank chat auto-deletion if unused
* Chat history management
* Seamless switching between conversations

## 8. User Progress Tracking

* Tracks daily learning activity
* Stores historical learning progress
* Personalized progress monitoring

## 9. Modern Responsive UI

* Desktop optimized interface
* Fully redesigned mobile-friendly UI
* Smooth sidebar interactions
* Professional modern dashboard design

## 10. Settings & Customization

* Mobile drawer-based settings panel
* Smooth animations and transitions
* AI provider selection interface
* Consistent UI across desktop and mobile

## 11. Progressive Web App (PWA)

* Installable like a mobile application
* Cross-device accessibility
* App-like user experience

## 12. Authentication System

* User signup and login system
* Personalized chat sessions
* User-specific learning data storage

## 13. Backend Infrastructure

* Built with Django framework
* PostgreSQL / Neon / Supabase database support
* Cloud deployment support
* Production server support using Gunicorn

---

# Project Goal

The main goal of **Uccharon** is to create an intelligent **AI-based English learning assistant** that helps users practice English daily, improve speaking confidence, expand vocabulary, and track long-term learning progress in a simple, interactive, and modern learning environment.

The platform is also designed to **analyze user responses**, **rate performance**, and provide **daily and weekly feedback** so learners can clearly understand their progress and continuously improve.
